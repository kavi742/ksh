# KSH
KSH (Kavi Shell) is a simple shell written in C
Based on [tutorial](https://github.com/brenns10/lsh) by Stephen Brennan

## What I Learned
- How shell languages interpret input and execute system calls
- String parsing

## Basic Functionality
- read
- parse
- exec
- fork
- wait
